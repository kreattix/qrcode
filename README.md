# QR Code Generator

A modern, feature-rich QR code generator built with Next.js, TypeScript, and Tailwind CSS. Create customizable QR codes for URLs, text, phone numbers, email, and Wi-Fi networks.

## Features

- Multiple QR code types:
  - URLs
  - Plain text
  - Email (with subject and body)
  - Phone numbers
  - Wi-Fi networks (with encryption options)
- Customizable colors (foreground and background)
- Live preview
- One-click download
- Dark mode support
- Fully responsive design
- SEO optimized

## Installation

1. Clone the repository:

```bash
git clone https://github.com/yourusername/qr-code-generator.git
cd qr-code-generator
```

2. Install dependencies:

```bash
npm install
```

## Development

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Dependencies

- Next.js 14.1.0
- React 18
- TypeScript 5
- Tailwind CSS 3
- qrcode.react 3.1.0
- react-colorful 5.6.1
- react-hot-toast 2.4.1
- @heroicons/react 2.1.1

## Building for Production

```bash
npm run build
```

Then start the production server:

```bash
npm start
```
